generate_data() function¶

yggdrasil_rapidjson.generate_data(schema)¶

Generate data that conforms to the provided schema.

Parameters:

dict (str,) – JSON schema to generate data for

Returns:

Python object that conforms to the provided schema

>>> generate_data('{"type": "number"}')
0.0
>>> generate_data({"type": "object", "properties": {"a": {"type": "integer"}}})
{'a': 0}
>>> generate_data({"type": "ndarray", "subtype": "complex"})
array([[0.+0.j, 0.+0.j],
       [0.+0.j, 0.+0.j]])
